Contributed Features and Support

  2016

Contributor to the Dota2 TheCore Config Engine, originally by loopuleasa and wrote additional features (ex. Quick Cast Micro), as well as providing voluntary support. In addition, I have written an installer to help simplify the instalation process.
Unfortuantly due to changes to the Dota2 Configuration Files, this project was broken and cannot be fixed without external tools. Thus it was discontinued.
